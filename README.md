# 2025电赛E题-云台控制
本项目为2025全国大学生电子设计竞赛E题云台控制部分的STM32实现，基于Keil开发环境。

## 项目结构
- `Hardware/`：硬件相关驱动代码
- `Library/`：STM32标准库文件
- `System/`：系统通用功能（延时、串口等）
- `User/`：用户主逻辑代码（main.c等）
- `Project.uvprojx`：Keil工程文件

## 开发环境
- 开发工具：Keil uVision5
- 目标平台：STM32系列单片机
- 编程语言：C语言

## 编译与运行
1.  用Keil打开`Project.uvprojx`工程文件
2.  配置对应单片机型号与下载器
3.  编译工程并下载到硬件，即可运行云台控制程序
